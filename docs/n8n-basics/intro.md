# 序言：为什么学 n8n

> 🚀 **玩转 n8n：小白也能学会的自动化**

## 这本书适合谁

- **完全小白**：没有编程基础，但想让重复工作自动化
- **白领上班族**：每天被日报、周报、数据整理折磨
- **中小企业主**：想提效降本，但请不起开发团队
- **跨境电商/外贸人**：需要对接多平台、多时区的自动化需求

## 为什么选择 n8n

| 特点              | 说明                     |
| ----------------- | ------------------------ |
| 🆓 **开源免费**   | 自己部署，数据在自己手里 |
| 🎨 **可视化拖拽** | 像画流程图一样搭建自动化 |
| 🌍 **400+ 集成**  | 支持主流国内外平台       |
| 🤖 **AI 友好**    | 遇到代码让 AI 帮你写     |

## 你将学到什么

1. **从零开始部署 n8n** - Docker 一键搞定
2. **核心概念扎实掌握** - 节点、触发器、表达式
3. **实战案例拿来即用** - 钉钉、微信、数据同步
4. **进阶技巧突破瓶颈** - 错误处理、生产部署

## 开始之前

你只需要准备：

- 一台能上网的电脑
- 一点点耐心
- 愿意动手尝试的心态
