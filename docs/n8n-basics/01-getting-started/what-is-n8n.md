# n8n 是什么

> 📌 **本节目标**：了解 n8n 的定位和核心价值

## 一句话介绍

**n8n** 是一个开源的工作流自动化工具，让你通过可视化拖拽的方式，把各种应用和服务串联起来，实现自动化。

## 它能做什么

举几个例子：

- 📮 **邮件自动转发** - 收到特定邮件，自动转发到钉钉群
- 📊 **数据自动汇总** - 每天自动从多个表格汇总数据，生成日报
- 🛒 **订单自动通知** - 客户下单后，自动发短信/微信通知
- 🤖 **AI 自动回复** - 接收消息，调用 AI 处理后自动回复

## 和其他工具的对比

| 工具    | 开源 | 自托管 | 中文支持 | 费用 |
| ------- | ---- | ------ | -------- | ---- |
| **n8n** | ✅   | ✅     | 社区翻译 | 免费 |
| Zapier  | ❌   | ❌     | ❌       | $$$$ |
| Make    | ❌   | ❌     | ❌       | $$$  |

## 为什么选 n8n

1. **数据安全** - 部署在自己服务器，敏感数据不外泄
2. **无限任务** - 不按执行次数收费
3. **高度可定制** - 可以写代码扩展功能
