# 让 AI 帮你写代码

> 📌 **本节目标**：学会用 AI 解决 n8n 中的代码问题

## 为什么需要代码

虽然 n8n 大部分操作都是拖拽完成，但有时还是需要写点代码：

- 复杂的数据转换
- 特殊的逻辑判断
- 自定义函数

## AI 帮你写

**不用怕！把需求告诉 AI，让它帮你生成代码。**

### 示例对话

```
我：我需要一段 n8n Code 节点的代码，
    把数组中的对象按照 createdAt 字段倒序排列

AI：可以使用这段代码...
```

## 常用 AI 工具

| 工具     | 特点           |
| -------- | -------------- |
| ChatGPT  | 通用能力强     |
| Claude   | 代码质量高     |
| DeepSeek | 中文友好，免费 |

## 写代码的技巧

1. **说清楚输入输出** - 告诉 AI 你的数据长什么样
2. **给具体例子** - 举例说明想要什么效果
3. **分步骤提问** - 复杂问题拆解成小问题

---

> 🎯 **记住**：零代码不是真的一行代码都不写，而是不需要你自己从头写！
