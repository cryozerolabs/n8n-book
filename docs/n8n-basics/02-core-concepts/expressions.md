# 表达式 (Expressions)

> 📌 **本节目标**：学会用表达式动态获取和处理数据

## 什么是表达式

表达式让你可以在节点配置中动态引用数据，而不是写死固定值。

## 基本语法

用双花括号 `{{ }}` 包裹表达式：

```
{{ $json.name }}
```

## 常用表达式

| 表达式                             | 作用               |
| ---------------------------------- | ------------------ |
| `{{ $json.字段名 }}`               | 获取当前数据的字段 |
| `{{ $('节点名').item.json.字段 }}` | 获取指定节点的数据 |
| `{{ $now }}`                       | 当前时间           |
| `{{ $today }}`                     | 当前日期           |

## 示例

发送邮件时动态填入收件人：

```
收件人：{{ $json.email }}
主题：您好，{{ $json.name }}
```
