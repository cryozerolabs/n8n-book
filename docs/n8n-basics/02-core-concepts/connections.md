# 连接与数据流

> 📌 **本节目标**：理解节点之间如何传递数据

## 连接的意义

连接线不仅表示执行顺序，还承载着数据的传递。

```
[触发节点] → [处理节点] → [输出节点]
              ↓
           数据流动
```

## 数据结构

n8n 中的数据以 **JSON** 格式流动：

```json
[
  {
    "json": {
      "name": "张三",
      "email": "zhangsan@example.com"
    }
  }
]
```

## 多输入输出

- 一个节点可以有多个输入
- 一个节点可以有多个输出
- 数据会自动合并或分发
