# 触发器 (Triggers)

> 📌 **本节目标**：了解工作流的启动方式

## 触发器是什么

触发器决定了工作流**何时**开始执行。没有触发器，工作流只能手动运行。

## 常用触发器

| 触发器            | 用途          | 场景                |
| ----------------- | ------------- | ------------------- |
| **Schedule**      | 定时执行      | 每天早上 9 点发日报 |
| **Webhook**       | HTTP 请求触发 | 接收外部系统回调    |
| **Email Trigger** | 收到邮件触发  | 邮件自动处理        |
| **Chat Trigger**  | 聊天消息触发  | 机器人对话          |

## 定时触发示例

设置每天早上 9:00 执行：

1. 添加 **Schedule Trigger** 节点
2. 选择 **Cron** 模式
3. 输入：`0 9 * * *`

---

> ⚠️ **注意**：一个工作流可以有多个触发器
