# 节点 (Nodes)

> 📌 **本节目标**：理解 n8n 中最核心的概念——节点

## 什么是节点

节点是 n8n 工作流的基本组成单元。每个节点执行一个特定的任务，比如：

- 📥 获取数据
- 🔄 转换数据
- 📤 发送数据

## 节点类型

| 类型         | 说明         | 示例                |
| ------------ | ------------ | ------------------- |
| **触发节点** | 启动工作流   | Schedule、Webhook   |
| **操作节点** | 执行具体操作 | HTTP Request、Gmail |
| **逻辑节点** | 控制流程     | IF、Switch          |
| **数据节点** | 处理数据     | Set、Code           |

## 添加节点

1. 点击画布中的 **"+"** 或拖拽连线
2. 在弹出的面板中搜索节点
3. 点击添加到画布
